Small pill for status and meta labels — the contest uses it for 필수 / 선택 / 완료 / 중복.

```jsx
<Badge variant="accent">필수</Badge>
<Badge variant="neutral">선택</Badge>
<Badge variant="success" icon="check">제출 완료</Badge>
<Badge variant="info" icon="info">이미 신청</Badge>
```

Variants: neutral · accent · success · warning · danger · info · outline. Sizes: `sm` (default) · `md`. `icon` accepts an Icon name or node.
